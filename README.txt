Для работы с COM-библиотеками на языке программирования GoLang используется дополнительная библиотека “go-ole”. Через методы этой библиотеки возможно подключение COM-библиотеки и использование полного ее интерфейса.
Для ее использования просто скопируйте папку "go-ole" в исходную папку ../Go/

Данный пример работает только под управлением ОС Windows.
